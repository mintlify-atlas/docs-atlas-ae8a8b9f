# Introduction

The Hedera DX Starter Kit is a production-ready Next.js template for building modern Hedera dApps with wallet integration and Mirror Node API access. Get started in minutes with a complete frontend stack that showcases real-time data from the Hedera network.

<Note>
This starter kit is a modern evolution of Hedera's official WalletConnect tutorial, rebuilt from the ground up with Next.js 15, TypeScript, React 19, and shadcn/ui.
</Note>

## Key features

The Hedera DX Starter Kit provides everything you need to build a production-ready Hedera dApp:

- **Next.js 15** - Latest App Router with React Server Components
- **TypeScript** - Full type safety across your entire application
- **React 19** - Cutting-edge React features and performance
- **Wallet integration** - MetaMask and HashPack support via WalletConnect
- **Mirror Node API** - Fast, scalable access to Hedera network data
- **shadcn/ui** - Beautiful, accessible components built with Radix UI and Tailwind CSS
- **Dark mode** - Built-in theme support with next-themes
- **Storybook** - Component development and documentation

## Architecture

The Hedera DX Starter Kit uses a client-first architecture that delivers instant performance:

- **No backend required** - All data fetched directly from Mirror Node REST API
- **Wallet-based security** - Private keys never leave your wallet
- **Fast and scalable** - Mirror Node handles all network queries
- **Zero deployment friction** - Deploy to Vercel, Netlify, or any static host

```typescript
import { MirrorNodeService } from '@/services/MirrorNodeService';

const mirror = MirrorNodeService.getInstance();
const info = await mirror.getAccountInfo('0.0.1234');
const tokens = await mirror.getAccountTokenBalancesWithInfo('0.0.1234');
const txs = await mirror.getAccountTransactions('0.0.1234', 10);
```
