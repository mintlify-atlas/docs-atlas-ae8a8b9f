The `AccountDetails` component displays comprehensive account information fetched from the Hedera Mirror Node. It shows account balances, token holdings, and transaction history in a tabbed interface.

## Overview

Location: `src/components/AccountDetails.tsx`

This component provides three main views:
1. **Overview**: Account balance, token count, and metadata
2. **Tokens**: Detailed list of all HTS token balances
3. **History**: Recent transaction history

## Usage

```tsx
import { AccountDetails } from '@/components/AccountDetails';

export default function Page() {
  return (
    <div>
      <AccountDetails />
    </div>
  );
}
```

The component automatically fetches data for the connected wallet account.

## Props

The `AccountDetails` component does not accept any props. It reads wallet state from `WalletContext` and fetches data automatically.

## Data Fetching

The component uses `MirrorNodeService` to fetch data:

```typescript
const fetchAccountData = useCallback(async () => {
  if (!wallet.isConnected || !wallet.accountId) return;
  setLoading(true);
  
  try {
    const results = await Promise.allSettled([
      mirrorNodeService.getAccountInfo(wallet.accountId),
      mirrorNodeService.getAccountTokenBalancesWithInfo(wallet.accountId),
      mirrorNodeService.getAccountTransactions(wallet.accountId, 10),
    ]);

    // Handle each result...
  } catch (error) {
    console.error('Error fetching account data:', error);
  } finally {
    setLoading(false);
  }
}, [wallet.isConnected, wallet.accountId]);
```

### Error Handling

The component uses `Promise.allSettled()` to handle partial failures gracefully:

- If account info fails, shows fallback data with connected account ID
- If token balances fail, shows empty array
- If transactions fail, shows empty array
- Errors are logged to console but don't break the UI

## Helper Functions

### Timestamp Formatting

```typescript
const safeTimestampToDate = (ts: string | number | undefined) => {
  if (!ts) return 'Unknown';
  const t = typeof ts === 'string' && ts.includes('.') 
    ? Math.floor(Number(ts.split('.')[0])) 
    : Number(ts);
  if (Number.isNaN(t) || t <= 0) return 'Unknown';
  return new Date(t * 1000).toLocaleString();
};
```
