The Hedera DX Starter Kit includes a comprehensive set of React components built with Next.js 15 and TypeScript. These components provide ready-to-use UI elements for building Hedera dApps.

## Component Foundation

All components are built on a solid foundation:

- **shadcn/ui philosophy**: Components follow shadcn/ui patterns - they're owned by you, not installed as dependencies
- **Tailwind CSS**: Styled with Tailwind utility classes for easy customization
- **TypeScript**: Fully typed for better developer experience
- **Accessibility**: Built with semantic HTML and ARIA labels
- **Dark mode**: All components support light/dark themes via `next-themes`

## UI Components

The starter kit includes foundational UI components in `src/components/ui/`:

- **Button**: Flexible button component with multiple variants (`default`, `outline`, `ghost`) and sizes
- **ThemeToggle**: Toggle between light and dark themes

## Storybook Integration

All major components include Storybook stories for interactive development and documentation:

```bash
npm run storybook
```

Storybook stories are available for:
- `WalletConnect`
- `AccountDetails`
- `ApiPlayground`
- `TokenRow`

This allows you to develop and test components in isolation before integrating them into your application.

## Component Architecture

### State Management

Components use React Context for global state:

- `WalletContext`: Manages wallet connection state across the app
- `ThemeProvider`: Handles theme switching (light/dark mode)

### Service Integration

Components integrate with service layers:

- `MirrorNodeService`: Fetches account data, token balances, and transactions
- Wallet providers: Handle connection to MetaMask and WalletConnect

### Styling Utilities

The `cn()` utility function (from `tailwind-merge` and `clsx`) is used throughout for conditional className merging:

```typescript
import { cn } from '@/lib/utils';

className={cn(
  'base-classes',
  conditionalClass && 'conditional-classes',
  className
)}
```
