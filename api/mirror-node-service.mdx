The `MirrorNodeService` is a singleton class that provides a client-side interface to the Hedera Mirror Node REST API. It allows you to query account information, token balances, transactions, and other on-chain data without executing transactions.

## Singleton Pattern

### getInstance()

Get the singleton instance of MirrorNodeService.

```typescript
static getInstance(): MirrorNodeService
```

<ResponseField name="instance" type="MirrorNodeService">
  The singleton instance of MirrorNodeService
</ResponseField>

**Example:**

```typescript
import { MirrorNodeService } from '@/services/MirrorNodeService';

const mirrorNode = MirrorNodeService.getInstance();
```

## Account Methods

### getAccountInfo()

Retrieve detailed account information from the Mirror Node. Supports both Hedera account IDs (0.0.xxxxx) and EVM addresses (0x...).

```typescript
async getAccountInfo(accountId: string): Promise<AccountInfo>
```

<ParamField path="accountId" type="string" required>
  Hedera account ID (e.g., "0.0.12345") or EVM address (e.g., "0x...")
</ParamField>

<ResponseField name="response" type="AccountInfo">
  Account information object containing balance, timestamps, and configuration
</ResponseField>

**Example:**

```typescript
const accountInfo = await mirrorNode.getAccountInfo('0.0.12345');
console.log(`Balance: ${accountInfo.balance.balance} tinybars`);
console.log(`EVM Address: ${accountInfo.evm_address}`);
```

### getAccountTokenBalances()

Retrieve all token balances for a specific account.

```typescript
async getAccountTokenBalances(accountId: string): Promise<TokenBalance[]>
```

<ParamField path="accountId" type="string" required>
  Hedera account ID or EVM address
</ParamField>

<ResponseField name="response" type="TokenBalance[]">
  Array of token balance objects (up to 100 tokens)
</ResponseField>

**Example:**

```typescript
const balances = await mirrorNode.getAccountTokenBalances('0.0.12345');
balances.forEach(balance => {
  console.log(`Token ${balance.token_id}: ${balance.balance}`);
});
```

### getAccountTokenBalancesWithInfo()

Retrieve token balances enriched with full token metadata (name, symbol, decimals, etc.).

```typescript
async getAccountTokenBalancesWithInfo(
  accountId: string
): Promise<Array<TokenBalance & Partial<TokenInfo>>>
```

<ParamField path="accountId" type="string" required>
  Hedera account ID or EVM address
</ParamField>

<ResponseField name="response" type="Array<TokenBalance & Partial<TokenInfo>>">
  Array of token balances merged with token information
</ResponseField>

**Example:**

```typescript
const enrichedBalances = await mirrorNode.getAccountTokenBalancesWithInfo('0.0.12345');
enrichedBalances.forEach(token => {
  console.log(`${token.name} (${token.symbol}): ${token.balance}`);
  console.log(`Decimals: ${token.decimals}`);
});
```

async getTokenInfo(tokenId: string): Promise<TokenInfo>

<ParamField path="tokenId" type="string" required>

### isTokenAssociated()

Check if an account is associated with a specific token.

```typescript
async isTokenAssociated(accountId: string, tokenId: string): Promise<boolean>
```

<ParamField path="accountId" type="string" required>
  Hedera account ID or EVM address
</ParamField>

<ParamField path="tokenId" type="string" required>
  Token ID to check association for
</ParamField>

<ResponseField name="response" type="boolean">
  `true` if the account is associated with the token, `false` otherwise
</ResponseField>

**Example:**

```typescript
const isAssociated = await mirrorNode.isTokenAssociated('0.0.12345', '0.0.67890');
if (!isAssociated) {
  console.log('Account needs to associate with token first');
}
```

## Transaction Methods

### getAccountTransactions()

Retrieve transaction history for an account, ordered by most recent first.

```typescript
async getAccountTransactions(
  accountId: string,
  limit: number = 25
): Promise<Transaction[]>
```

<ParamField path="accountId" type="string" required>
  Hedera account ID or EVM address
</ParamField>

<ParamField path="limit" type="number" default={25}>
  Maximum number of transactions to retrieve (default: 25)
</ParamField>

<ResponseField name="response" type="Transaction[]">
  Array of transaction objects in descending chronological order
</ResponseField>

**Example:**

```typescript
const transactions = await mirrorNode.getAccountTransactions('0.0.12345', 50);
transactions.forEach(tx => {
  console.log(`${tx.name} - ${tx.result}`);
  console.log(`Transaction ID: ${tx.transaction_id}`);
  console.log(`Fee: ${tx.charged_tx_fee} tinybars`);
});
```

### getTransactionStatus()

Convert a transaction result code into a human-readable status message.

```typescript
getTransactionStatus(result: string): { success: boolean; message: string }
```

<ParamField path="result" type="string" required>
  Transaction result code (e.g., "SUCCESS", "INSUFFICIENT_PAYER_BALANCE")
</ParamField>

<ResponseField name="success" type="boolean">
  Whether the transaction was successful
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable status message
</ResponseField>

**Example:**

```typescript
const status = mirrorNode.getTransactionStatus('SUCCESS');
console.log(status.message); // "Transaction successful"

const errorStatus = mirrorNode.getTransactionStatus('INSUFFICIENT_PAYER_BALANCE');
console.log(errorStatus.message); // "Insufficient balance"
```

## Formatting Utilities

### formatHbar()

Convert tinybars to HBAR with 2 decimal places.

```typescript
formatHbar(tinybars: number): string
```

<ParamField path="tinybars" type="number" required>
  Amount in tinybars (1 HBAR = 100,000,000 tinybars)
</ParamField>

<ResponseField name="response" type="string">
  Formatted HBAR amount as string (e.g., "5.00")
</ResponseField>

**Example:**

```typescript
const hbar = mirrorNode.formatHbar(500000000);
console.log(`${hbar} HBAR`); // "5.00 HBAR"
```

### formatTokenAmount()

Format token amount with proper decimal places.

```typescript
formatTokenAmount(amount: number, decimals: number = 0): string
```

<ParamField path="amount" type="number" required>
  Raw token amount (smallest unit)
</ParamField>

<ParamField path="decimals" type="number" default={0}>
  Number of decimal places for the token
</ParamField>

<ResponseField name="response" type="string">
  Formatted token amount with locale-specific thousand separators
</ResponseField>

**Example:**

```typescript
const amount = mirrorNode.formatTokenAmount(1000000, 6);
console.log(amount); // "1"

const largeAmount = mirrorNode.formatTokenAmount(1234567890, 2);
console.log(largeAmount); // "12,345,678.90"
```

## Usage Example

```typescript
import { MirrorNodeService } from '@/services/MirrorNodeService';

const mirrorNode = MirrorNodeService.getInstance();

// Get account information
const accountInfo = await mirrorNode.getAccountInfo('0.0.12345');
const hbarBalance = mirrorNode.formatHbar(accountInfo.balance.balance);
console.log(`Account Balance: ${hbarBalance} HBAR`);

// Get token balances with metadata
const tokens = await mirrorNode.getAccountTokenBalancesWithInfo('0.0.12345');
tokens.forEach(token => {
  const formattedAmount = mirrorNode.formatTokenAmount(
    token.balance,
    parseInt(token.decimals || '0')
  );
  console.log(`${token.symbol}: ${formattedAmount}`);
});

// Check transaction history
const transactions = await mirrorNode.getAccountTransactions('0.0.12345', 10);
transactions.forEach(tx => {
  const status = mirrorNode.getTransactionStatus(tx.result);
  console.log(`${tx.name}: ${status.message}`);
});
```
