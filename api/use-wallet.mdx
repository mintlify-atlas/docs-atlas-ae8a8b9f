## Overview

The `useWallet` hook provides access to the wallet context, allowing components to read and update wallet state. It must be used within a `WalletProvider`.

**Source:** `src/contexts/WalletContext.tsx:40`

## Usage

```tsx
import { useWallet } from '@/contexts/WalletContext';

export function MyComponent() {
  const { wallet, setWallet, disconnectWallet } = useWallet();
  
  if (!wallet.isConnected) {
    return <div>Please connect your wallet</div>;
  }
  
  return (
    <div>
      <p>Account: {wallet.accountId}</p>
      <p>Balance: {wallet.balance} HBAR</p>
      <button onClick={disconnectWallet}>Disconnect</button>
    </div>
  );
}
```

## Return Value

The hook returns an object with the following properties:

<ResponseField name="wallet" type="WalletInfo" required>
  Current wallet state
  
  <Expandable title="WalletInfo properties">
    <ResponseField name="accountId" type="string">
      The Hedera account ID (e.g., "0.0.12345") or EVM address
    </ResponseField>
    
    <ResponseField name="balance" type="string">
      The HBAR balance as a string
    </ResponseField>
    
    <ResponseField name="isConnected" type="boolean">
      Whether a wallet is currently connected
    </ResponseField>
    
    <ResponseField name="provider" type="any" optional>
      The wallet provider instance (WalletConnect or MetaMask)
    </ResponseField>
    
    <ResponseField name="walletType" type="'walletconnect' | 'metamask'" optional>
      The type of wallet connected
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="setWallet" type="(wallet: WalletInfo) => void" required>
  Function to update the wallet state. Use this after successfully connecting a wallet.
  
  ```tsx
  setWallet({
    accountId: '0.0.12345',
    balance: '100.00',
    isConnected: true,
    provider: walletProvider,
    walletType: 'walletconnect'
  });
  ```
</ResponseField>

<ResponseField name="disconnectWallet" type="() => void" required>
  Function to disconnect the wallet and clear all stored data. Automatically clears localStorage items.
  
  ```tsx
  disconnectWallet(); // Resets wallet state and clears localStorage
  ```
</ResponseField>

## Type Definition

```typescript src/contexts/WalletContext.tsx:40-45
export function useWallet() {
  const context = useContext(WalletContext);
  if (context === undefined) {
    throw new Error('useWallet must be used within a WalletProvider');
  }
  return context;
}
```

## Error Handling

The hook will throw an error if used outside of a `WalletProvider`:

```typescript
if (context === undefined) {
  throw new Error('useWallet must be used within a WalletProvider');
}
```

Make sure your component tree is wrapped with `WalletProvider`:

```tsx
// ✅ Correct
<WalletProvider>
  <MyComponent /> {/* Can use useWallet */}
</WalletProvider>

// ❌ Incorrect - will throw error
<MyComponent /> {/* useWallet will throw */}
```
