## Overview

This page documents all exported TypeScript interfaces and types used throughout the Hedera DX Starter Kit. These types provide full type safety for Hedera account data, tokens, transactions, and wallet interactions.

## Wallet Types

### WalletInfo

Represents the connected wallet state.

<ResponseField name="accountId" type="string" required>
  The Hedera account ID (e.g., "0.0.12345") or EVM address (0x...)
</ResponseField>

<ResponseField name="balance" type="string" required>
  The HBAR balance as a string
</ResponseField>

<ResponseField name="isConnected" type="boolean" required>
  Whether a wallet is currently connected
</ResponseField>

<ResponseField name="provider" type="any" optional>
  The wallet provider instance (WalletConnect EthereumProvider or window.ethereum)
</ResponseField>

<ResponseField name="walletType" type="'walletconnect' | 'metamask'" optional>
  The type of wallet that is connected
</ResponseField>

#### Usage Example

```typescript src/components/WalletConnect.tsx:37-43
const walletInfo: WalletInfo = {
  accountId: accountId,
  balance: '0.00',
  isConnected: true,
  provider,
  walletType: 'walletconnect'
};
```

## Usage Examples

### Type-safe Account Data Fetching

```typescript src/components/AccountDetails.tsx:36-40
const [accountInfo, setAccountInfo] = useState<AccountInfo | null>(null);
const [tokenBalances, setTokenBalances] = useState<Array<TokenBalance & Partial<TokenInfo>>>([]);
const [transactions, setTransactions] = useState<Transaction[]>([]);

const results = await Promise.allSettled([
  mirrorNodeService.getAccountInfo(wallet.accountId),
  mirrorNodeService.getAccountTokenBalancesWithInfo(wallet.accountId),
  mirrorNodeService.getAccountTransactions(wallet.accountId, 10),
]);
```

### Accessing Nested Types

```typescript
const accountInfo: AccountInfo = await mirrorNodeService.getAccountInfo('0.0.12345');

// Access balance
const balanceInTinybars: number = accountInfo.balance.balance;
const balanceInHbar = balanceInTinybars / 100_000_000;

// Access optional fields safely
const evmAddress: string | undefined = accountInfo.evm_address;
if (evmAddress) {
  console.log(`EVM Address: 0x${evmAddress}`);
}
```

### Type Guards

```typescript
function isConnected(wallet: WalletInfo): wallet is WalletInfo & { isConnected: true } {
  return wallet.isConnected === true;
}

if (isConnected(wallet)) {
  // TypeScript knows wallet.isConnected is true here
  console.log(wallet.accountId);
}
```
