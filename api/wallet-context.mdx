## Overview

The `WalletContext` provides a centralized state management solution for wallet connectivity in your Hedera dApp. It manages wallet connection state, account information, and provides methods to connect and disconnect wallets.

**Source:** `src/contexts/WalletContext.tsx:12`

## WalletProvider

The `WalletProvider` component wraps your application and provides wallet state to all child components.

### Usage

Wrap your application with the `WalletProvider` in your root layout or providers file:

```tsx
import { WalletProvider } from '@/contexts/WalletContext';

export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <WalletProvider>
      {children}
    </WalletProvider>
  );
}
```

### Implementation

```tsx src/contexts/WalletContext.tsx:14-38
export function WalletProvider({ children }: { children: ReactNode }) {
  const [wallet, setWallet] = useState<WalletInfo>({
    accountId: '',
    balance: '',
    isConnected: false,
  });

  const disconnectWallet = () => {
    setWallet({
      accountId: '',
      balance: '',
      isConnected: false,
    });
    // Clear localStorage
    localStorage.removeItem('wallet_connected');
    localStorage.removeItem('wallet_account');
    localStorage.removeItem('wallet_type');
  };

  return (
    <WalletContext.Provider value={{ wallet, setWallet, disconnectWallet }}>
      {children}
    </WalletContext.Provider>
  );
}
```

## Methods

### disconnectWallet()

Disconnects the current wallet and clears all stored data including:
- Wallet state (accountId, balance, isConnected)
- localStorage items (wallet_connected, wallet_account, wallet_type)

```typescript src/contexts/WalletContext.tsx:21-31
const disconnectWallet = () => {
  setWallet({
    accountId: '',
    balance: '',
    isConnected: false,

  localStorage.removeItem('wallet_connected');
  localStorage.removeItem('wallet_account');
  localStorage.removeItem('wallet_type');

```

## Example: Connecting a Wallet

```tsx
import { useWallet } from '@/contexts/WalletContext';

export function MyWalletButton() {
  const { setWallet } = useWallet();
  
  const handleConnect = async () => {
    // After successful wallet connection
    setWallet({
      accountId: '0.0.12345',
      balance: '100.00',
      isConnected: true,
      provider: walletProvider,
      walletType: 'walletconnect'
    });
    
    // Persist to localStorage
    localStorage.setItem('wallet_connected', 'true');
    localStorage.setItem('wallet_account', '0.0.12345');
    localStorage.setItem('wallet_type', 'walletconnect');
  };
  
  return <button onClick={handleConnect}>Connect</button>;
}
```
